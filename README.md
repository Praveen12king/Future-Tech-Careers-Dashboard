# 📌 Project Overview

The **Future Tech Careers Dashboard** is an interactive **Tableau analytics dashboard** that explores the global technology job market.

This project analyzes technology careers, salary trends, industries, and global job distribution to help understand how the tech job market is evolving.

The dashboard enables users to explore:

• Global technology job demand
• Technology roles and career domains
• Industry demand for tech skills
• Job distribution across countries
• Trends in future technology careers

---

# 🎯 Project Objectives

The main objective of this project is to analyze the **global technology job market** and identify insights related to career opportunities.

Key goals include:

• Analyze global demand for technology jobs
• Identify the most in-demand tech roles
• Understand industry demand for technology skills
• Explore job distribution across countries
• Provide insights into future tech career opportunities

---

# 📊 Dashboard Features

## 1️⃣ Global Job Market Overview

The dashboard provides a high-level overview of the global technology job market.

### Key Metrics

• Total Jobs
• Total Countries
• Total Technology Domains
• Total Job Roles

### Visualizations

📊 Job Distribution by Country
📈 Technology Job Trends
📊 Industry-wise Job Demand
📊 Top Technology Roles

---

## 2️⃣ Technology Career Domains

The dashboard analyzes different technology domains such as:

• Data Science
• Artificial Intelligence
• Cloud Computing
• Cyber Security
• Software Development

Users can explore which technology fields have the highest demand.

---

## 3️⃣ Global Job Distribution

A world map visualization highlights the countries where technology job opportunities are available.

This helps identify:

• Countries with the highest demand for tech jobs
• Global hiring trends
• Emerging technology markets
